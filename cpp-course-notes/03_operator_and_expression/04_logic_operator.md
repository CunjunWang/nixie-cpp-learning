# 逻辑运算符
## 运算符
假设`A`值为`1`, `B`值为`0`

| 运算符 | 描述 | 结果 |
| --- | --- | --- |
| `&&` | 与逻辑, 两个都为真则为真 | `A && B`为`false` |
| `II` | 或逻辑, 有一个为真则为真 | `A II B`为`true` |
| `!` | 非逻辑, 取反 | `!(A && B)`为`true` |

## 示例
```
cout << (A == true || A != true) << endl; // 1
```

### 德摩根定律
```
!(A || B) = !A && !B
!(A && B) = !A || !B
```
