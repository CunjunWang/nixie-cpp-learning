# 算数运算符
## 算数运算符
假设`int A = 10, int B = 20`

| 运算符 | 描述 | 结果 |
| --- | --- | --- |
| `+` | 两个操作数相加 | `30` |
| `-` | 第一个操作数减去第二个操作数 | `-10` |
| `*` | 两个操作数相乘 | `200` |
| `/` | 第一个操作数除以第二个操作数 | `B / A = 2` <br> `A / B = 0` |
| `%` | 取模 | `B % A = 0` |
| `++` | 自增, 整数值增加`1` | `++A = 11` |
| `--` | 自减, 整数值减少`1` | `--A = 9` |
