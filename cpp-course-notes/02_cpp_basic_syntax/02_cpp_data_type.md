# C++数据类型
引例:
```
int sum;     // 定义整形变量
int m = 1;   // 定义整形变量
int n = 1;   // 定义整形变量
sum = m + n; // 求和赋值
```

## 数据类型的内部结构
- C++中每个变量都有其数据类型. 数据类型决定变量所占的内存空间大小和布局方式、该空间能存储的值和范围, 以及变量能参与的运算.
- C++是强类型语言
